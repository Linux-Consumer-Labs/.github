# linux consumer labs

**Researching ways to reach "Just Works" on the Linux desktop.**

We explore what it would take for Linux to feel as reliable and effortless as the best consumer operating systems — by asking what-ifs, learning from other platforms, and working *with* the kernel instead of against it.

- **Exploring What-Ifs**
- **Learning from Other OSes**
- **Not Strictly Following Linux Practices**
- **Working Alongside the Kernel**
